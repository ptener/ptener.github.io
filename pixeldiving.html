Pixel Diving

Building the ultimate mobile diving game

Pixel Diving's Development

I am a diver at Indiana University - Purdue University at Indianapolis, or IUPUI for short. I have always loved mobile diving games, but I could never find one that had everything that I wanted to have as a diver. This game is going to be my attempt at making the best diving game that I am capable of making.

I can't make it the best game all by myself though, so I will be using this page to get feedback from other divers/game developers on how the game is doing and things I should add in the future. I hope this game can become the best diving game made for divers by divers!

Current Progress

I am in the progress of moving Pixel Diving from the Godot to Unity game engine. My reason for this is beacase I have more expiriance with Unity as a game engine, and Unity will be able to more easily support the future plans I have for Pixel Diving.
